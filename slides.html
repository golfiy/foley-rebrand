Internal · style

Brand quick reference

Slide title font is Inter Tight, sentence case (no ALL CAPS).

  • Slide body copy font is Inter, regular weight.
  • Space for text on slides is optional — not every text box has to be used.
  • When a slide has multiple text blocks, keep the text size consistent across them.
  • Use brand colors only — do not use arbitrary colors without approval from marketing.
  • Foley Blue is the only brand accent. Green is reserved for "cleared" status. Red is for errors only.
